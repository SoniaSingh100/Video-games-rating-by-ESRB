🎮 Video Game Rating Prediction (ESRB)

A data-driven machine learning project that analyzes and predicts ESRB (Entertainment Software Rating Board) ratings of video games based on content features.

📌 Overview

This project focuses on understanding patterns in video game ratings and building a classification model to predict ratings such as:

E (Everyone)
E10+ (Everyone 10+)
T (Teen)
M (Mature)

Using Exploratory Data Analysis (EDA) and machine learning techniques, the project uncovers trends in game content and audience targeting.

🚀 Key Highlights
📊 Conducted EDA on 1900+ ESRB-rated video games
🔍 Identified trends in rating distribution and content descriptors
🤖 Built a multi-class classification model
🎯 Achieved 86.54% accuracy
📈 Strong precision & recall across all rating categories
